🔗 كيف تربط جهازين؟

عشان التغييرات تنعكس فوراً على جهازك الثاني، لازم تربطهم بقاعدة بيانات مشتركة عبر Firebase (مجاني من Google).

📋 خطوات الإعداد (مرة وحدة بس):

  1. افتح console.firebase.google.com من الكمبيوتر
  2. اضغط "إنشاء مشروع" وسمّيه مثلاً "عائلتي"
  3. من القائمة اليسرى: Realtime Database → إنشاء قاعدة بيانات
  4. اختر المنطقة الأقرب (مثلاً europe-west1)
  5. في "القواعد" غيّرها ل:
{ "rules": { ".read": true, ".write": true } }
  1. من ⚙️ إعدادات المشروع → General
  2. انسخ رابط قاعدة البيانات (يبدأ بـ https://...firebaseio.com)
  3. لصقه هنا ↓
